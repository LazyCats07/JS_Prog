Angka tanpa desimal 
-> 10, 1500, 123456
-> akurat sampai 15 digital

Angka dengan desimal 
-> 3.14, 0.5, 100.00

Eksponen
-> 123e5 // 12300000
-> 123e-5 // 0.00123

Bilangan Negatif 
-> 022 // 18 -> akan dianggap sebagai bilangan oktal 
-> 0xFF // 255 -> akan dianggap sebagai bilangan hexadesimal 

Angka Spesial 
-> 2 / 0 // Infinty
-> -2 / 0 // -Infinity
-> 0 / 0 // NAN (Not a Number)
-> 100 / "apel" // NaN (Not a Number)

Hati-Hati
100 / "10" // 10
